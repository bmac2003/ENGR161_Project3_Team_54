# ENGR161_Project3_Team_54

Contains all project test and runnable files for ENGR161 Final Project
Objective: Design an autonomous robot to follow a line, traverse obstacles, navigate based on magnetic markers, and unload cargo at specified points.  The robot was created using RaspberryPi infrastructure with LEGO Mindstorm motors and GrovePi sensors, and programmed using Python 3.
Project Length: September 4, 2021 - December 17, 2022

Main test code is found within POC Code listed files, with other programs designed to test specific functionality with limited scope. 
For finalized, commented code, view _POC Code v5_
